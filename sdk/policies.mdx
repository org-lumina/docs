## `lumina.policies.list()`

```ts
const policies = await lumina.policies.list()
//   [{ policyId, productId, buyer, coverageAmount, premiumPaid, txHash, status, … }]
```

Filtered server-side to the calling wallet. Cross-wallet listing returns 403.

## `lumina.policies.get(productId, policyId)`

Public — no API key required.

```ts
const p = await lumina.policies.get(
  '0xdc5bcc7d…',         // bytes32 productId
  '42'                    // policyId
)
```

## `lumina.policies.purchase(params)`

SDK 0.3.0+ auto-resolves both the `productId` keccak hash AND the per-shield
`asset` literal from `productName`. Pass `productName` whenever you can:

```ts
const receipt = await lumina.policies.purchase({
  productName: 'FLASHBTC1H-001',  // resolves productId + asset='BTC'
  buyer: '0x…',
  coverageAmount: '50000000',     // string of USDC base units
  idempotencyKey: 'uuid-v4',      // optional but recommended
})
```

Or pass `productId` explicitly — the SDK still auto-resolves the asset by
reverse-lookup against the canonical registry:

```ts
const receipt = await lumina.policies.purchase({
  productId: '0xe87625ef…',       // FLASHBTC1H-001 hash
  buyer: '0x…',
  coverageAmount: '50000000',
})
```

Returns:

```ts
{
  txHash: '0x…',
  blockNumber: 12345,
  policyId: '7',
  buyer: '0x…',
  productId: '0x…',
  coverageAmount: '50000000',
  premiumPaid: '450000',         // 6-dec USDC base units
}
```

### Asset encoding

The `asset` field is **optional** when `productName` or a registered
`productId` is supplied. To override, pass a symbol (`'BTC' | 'ETH' | 'USDC'
| 'USDT'`) or a 32-byte hex string:

```ts
asset: 'BTC'    // encoded to 0x4254430000…0000 for you
asset: '0x5553444300000000000000000000000000000000000000000000000000000000'
```

See [Products and assets](/agents/products-and-assets) for the full per-shield
table and a list of `InvalidAsset` failure modes.

### Idempotency

Always pass `idempotencyKey` on retryable purchases. The header value is
forwarded as `Idempotency-Key`. Replays return the original response; no
double-charge.
