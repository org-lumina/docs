This is the shortest path from zero to a real on-chain policy. Five minutes if
your wallet is funded with Base Sepolia ETH (for the onboarding signature only —
the protocol itself never asks the agent for gas).

## 1. Install the SDK

```bash
npm install @lumina-org/sdk ethers
```

## 2. Mint an API key (self-service)

Sign one message with your wallet — the API verifies and returns a
fresh `lk_…` key.

```ts
import { Wallet } from 'ethers'
import { LuminaClient } from '@lumina-org/sdk'

const wallet = new Wallet(process.env.PRIVATE_KEY!)
const lumina = new LuminaClient({ apiKey: '' })  // empty for onboard

const { apiKey } = await lumina.agent.onboard(wallet, { label: 'my-bot' })
console.log('Save this NOW (shown only once):', apiKey)
```

The signed message is `Lumina onboarding for {address} at {timestamp}`. The
SDK builds it for you. Caps: 3 active keys per wallet, 10 onboard attempts
per hour per IP, ±5 minute timestamp window.

## 3. Verify connectivity

```ts
const lumina = new LuminaClient({ apiKey: process.env.LUMINA_API_KEY! })
const health = await lumina.health()
console.log('chainId:', health.chain.chainId)              // 84532
console.log('contracts:', Object.keys(health.contracts))   // coverRouter, …
```

## 4. Buy a $100 policy

```ts
const policy = await lumina.policies.purchase({
  productName: 'FLASHBTC1H-001',                // SDK 0.3.0+ auto-resolves productId hash + asset='BTC'
  buyer: '0xYourWalletAddress',                 // wallet that pays USDC premium
  coverageAmount: '100000000',                  // $100 in 6-dec USDC (on-chain minimum)
})

console.log('policyId:', policy.policyId, 'tx:', policy.txHash)
```

That's it. The relayer paid the gas, your buyer wallet paid the USDC premium,
and you got back a `policyId` you can reference for triggers and bonds.

The SDK derives both the `productId` keccak hash and the per-shield `asset`
literal from `productName`. See [Products and assets](/agents/products-and-assets)
for the full registry and why hardcoding `asset: 'USDC'` reverts 7-of-9 shields.

## Try it without an account

If you don't even want to go through onboarding, hit the sandbox:

```bash
curl -X POST https://lumina-api-production-ac85.up.railway.app/sandbox/try
```

Server-fixed buyer + `$1` coverage. Useful for first-call demos. 10 req/h/IP.
